Creating an MCP Server

An MCP server exposes resources and tools to AI clients. Here's a practical example using Node.js:

import { Server } from '@modelcontextprotocol/sdk/server/index.js'; import { StdioServerTransport } from '@modelcontextprotocol/sdk/server/stdio.js'; class DatabaseMCPServer { constructor() { this.server = new Server({ name: "database-server", version: "1.0.0" }, { capabilities: { resources: {}, tools: {} } }); this.setupHandlers(); } setupHandlers() { // Resource handlers this.server.setRequestHandler('resources/list', async () => ({ resources: [ { uri: "postgresql://users", name: "User Database", mimeType: "application/json" } ] })); // Tool handlers this.server.setRequestHandler('tools/list', async () => ({ tools: [ { name: "query_database", description: "Execute SQL queries safely", inputSchema: { type: "object", properties: { query: { type: "string" }, params: { type: "array" } } } } ] })); this.server.setRequestHandler('tools/call', async (request) => { const { name, arguments: args } = request.params; if (name === "query_database") { // Implement safe query execution return await this.executeQuery(args.query, args.params); } }); } }

Best Practices

Error Handling: Implement comprehensive error handling with meaningful error messages

Input Validation: Always validate and sanitize inputs, especially for database queries

Resource Management: Properly manage connections and clean up resources

Documentation: Provide clear descriptions for all tools and resources

Client Integration

Integrating with MCP servers from client applications:

import { Client } from '@modelcontextprotocol/sdk/client/index.js'; const client = new Client({ name: "my-ai-app", version: "1.0.0" }); // Connect to MCP server await client.connect(transport); // List available tools const tools = await client.request('tools/list', {}); // Call a tool const result = await client.request('tools/call', { name: "query_database", arguments: { query: "SELECT * FROM users WHERE active = ?", params: [true] } });
